◆ create_mat_op()

vector<vector<poly> > create_mat_op ( int  nb_bits)

For computing the precomputation data : Create a matrix with 3 columns where each row contains 2 linear operations and the quadratic operation obtained by performing a AND beetween these 2 functions. For example, the line 10|13|61 means : (10 = Ob1010 = x0+x2) * (13 = Ob1101 = x0+x1+x3) = (x0 + x0x1 + x0x3 + x0x2 + x1x2 + x2x3 = Ob111101 = 61).
